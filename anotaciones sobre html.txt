h1: titular mas importante
h2: titular importante
h3,h4,h5 etc

a: etiqueta de enlace, <a href:"https://">: declaración del enlace, todo dentro de esta etiqueta es un enlace
atributo href: el enlace en sí
img src="": Imagen, tambien se puede poner una imagen como enlaces
style ="width": edita o reescala los pixeles de la imagen, puede ir sin el atributo style, puede ser en XXXpx o en XXX%
style ="height": edita solo la altura de los pixeles, puede ir sin el atributo style
elementos en html: es todo el contenido

br: Separa en lineas o salto de linea
hr: Separar secciones con lineas
pre: Parrafo como esta escrito en codigo

b: Ennegreser una palabra
strong: Significado más importante, hace lo mismo que etiqueta b

i: Texto en cursiva
em: Importancia semántica, texto cursivo

small: Disminuye la fuente
mark: Resaltar texto
del: Texto eliminado
ins: Texto insertado, subraya el texto
sub: subíndice
sup: Superíndice
<!-->: comentario
target = "_blank" : abrira el enlace en una nueva pestaña
mailto: gestor de emails
alt: texto alternativo, sirve como validador y sirve para cuando una imagen no carga

Tablas

<table>: generar tablas
tr: filas
th: encabezado de la tabla
td: columnas normales
colspan = x : Cantidad de celdas a ocupar, para columnas
rowspan = x : Cantidad de celdas a ocupar, para filas
ul: generar listas desordenadas, se pueden tener listas anidadas (listas dentro de listas)
ol: generar listas ordenadas
li: parametro de listas
start: que numero iniciar la lista

Formularios

form: define un formulario
input: ingresa variables, ej <input type="submit"> boton de enviar formulario
action: Define la acción a realizar cuando se envia el formulario
target: especifica que cuando se envie el formulario se cambiara de pagina o no, _self en la misma ventana, _blank en otra pestaña
method: el metodo por defecto "get" o "post", el metodo get es publico y post para info privada
name: identificar el dato
fieldset: Agrupar datos relacionados al formulario
legend: Titulo al fildset
novalidate: especifica si el navegador tiene que validar el formulario
id: identificador del dato
label for="id": nombre de input
textarea: Escribir mucho texto
button: generar botones
datalist: generar una lista con diferentes opciones y/o sugerencias
reset: resetear el formulario

tipos de inputs

radio: seleccionar 1 opcion definida
checkbox: elegir multiples opciones
color: seleccionar diversos colores
date: fechas
datetime-locar: fechas y horas
file: elegir ficheros
month: seleccionar meses
number: datear numeros
range: dales un valor minimo min, y un valor maximo max
search: realizar busquedas
tel: Telefono
time: Poner hora
url: valida si es una url
week: seleccionar semanas

atributos de inputs

value: titulo por asi decirlo
readonly: solo lectura
disabled: desabilita la opcion
size: numero de caracteres que caben en la casilla
maxlength: numero de caracteres que se pueden utilizar
<input type="file" multiple>: seleccionar varios ficheros
pattern: tipo de expresion regular {n}, donde n es la cantidad de letras
placeholder: nombre de la casilla
required: obligacion de rellenar
autofocus: casilla por defecto

selectores
select: da a elegir opciones
option: define las opciones
selected: valor por defecto
size: tamaño
multiple: Seleccionar varias cosas en el selector

videos y audio

<video controls>: insertar video
source src="" type="video/mp4": direccion del video
autoplay: el video se reproduce al entrar a la pag
<audio controls>: insertar audio
source src="" type="audio/mpeg: direccion del audio
autoplay: audio se reproduce al entrar
mute: sonido por defecto quitado

incrustar videos de youtube

opcion embed de youyube copiarla en el codigo, se puede agregar variables como controls, autoplay, mute, etc

citaciones

<q>: para crear citas cortas
blockquote: define una cita que viene de otra fuente(pag)
abbr: define abreviacion o acronimo <abbr tittle="xxxx">n</abbr>, n es el acronimo y xxxx lo que significa
address: define info de contacto
cite: titulo de una obra
bdo: texto de izquierda a derecha

bloques y elementos en linea

div: Se usa para meter dentro otros elementos
span: se usa para cosas mas concretas dentro de una linea

elementos de codigo

kbd: entrada por parte del usuario, ya sea por teclado o por voz
samp: outputs de programas
code: codigo de computadora
var: etiqueta para variables, sea matematico o de otro tipo

Entidades

&entity_name;
&nbsp: non-breaking space

iframe

<iframe>: permite poner atributo src a una ruta

css
border: bordes de tablas, border = none: sin bordes
border-collapse: Junta las lineas de los bordes
padding: Da un margen o una separacion, se declara en px
